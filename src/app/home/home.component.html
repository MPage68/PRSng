PRS (Purchase Request System)

Capstone

The purchase request system (PRS) project was completed by the Java Fall 2018 boot camp at MAX Technical Training/Cincy Code IT. The project utilizes several programs that were learned in the class, including:

Front-End: Back-End:
Angular Java
CSS Spring
HTML MySQL

The purchase request system generates requests for items needed by employees. The requests are then sent to purchasing for review. Once approved, a purchase order is generated and the items ordered. The PRS assists in control and tracking of purchases within the organization.
A user logs in to make a purchase request with their username and password. Once the user makes a purchase request, they can perform basic CRUD functions. The purchase request is then reviwed, approved, or rejected.
The PRS data base contains lists of products, users, and approved vendors.
